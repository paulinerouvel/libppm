## Launch Project

To use our library, **cargo** is required.

Compile the library by executing the command :

`cargo build`

You can then find the library in the `target/` folder

## Structs

### Pixel

To create a pixel, use the struct like so :

`let p = Pixel{r:255, g:255, b:255}`
